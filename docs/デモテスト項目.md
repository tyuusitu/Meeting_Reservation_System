# デモテスト項目

## 1. 目的

会議室追加、大会議室作成、大会議室予約時の占有判定、Googleカレンダー連携を含めて、会議室予約システム全体を再確認する。

確認日: 2026-05-07  
実行URL: `http://127.0.0.1:8087/`  
ブラウザ確認: Browser Use in-app browser  
対象: 静的フロント、GASコード、Googleスプレッドシート/Googleカレンダー連携設計

## 2. 追加仕様

| 機能 | 内容 |
|---|---|
| 会議室追加 | 管理画面から会議室名、種別、表示順、利用可否、カレンダーIDを保存する |
| カレンダー同時作成 | カレンダーIDが空の場合、GASがGoogleカレンダーを作成して会議室シートへ保存する |
| 大会議室作成 | 複数の単体会議室を構成会議室として選び、大会議室を作成する |
| 大会議室予約 | 大会議室を予約すると、構成会議室も同じ時間に使用中扱いにする |
| カレンダー反映 | 大会議室予約時は大会議室カレンダーと構成会議室カレンダーに予定を作成する |
