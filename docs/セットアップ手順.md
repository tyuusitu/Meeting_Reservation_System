## CONFIGの書き換え箇所

`config.js` の `window.APP_CONFIG = { ... }` を書き換えます。  
`gid` は任意ですが、入れるとそのシートは `export?format=csv&gid=...` を優先するので少し有利です。

```js
window.APP_CONFIG = {
  SPREADSHEET_ID: 'スプレッドシートID',   // URLの /d/【ここ】/edit
  GAS_API_URL:    'GAS Web APIのURL',       // デプロイ後のexec URL
  SHEET_GID: {
    reservations: '予約シートの gid',
    rooms: '会議室シートの gid',
    settings: '設定シートの gid',
    members: 'メンバー一覧シートの gid',
    meetings: '会議一覧シートの gid',
    attendanceResponses: '出欠回答シートの gid',
    meetingAggregations: '会議別集計シートの gid',
    memberAggregations: '個人別集計シートの gid',
    departmentAggregations: '部局別集計シートの gid',
    streaks: '連続欠席チェックシートの gid',
    logs: '操作ログシートの gid',
  },
};
```

`SHEET_GID` が空の項目は、`SHEET_NAME` のシート名読み込みへ自動フォールバックします。
