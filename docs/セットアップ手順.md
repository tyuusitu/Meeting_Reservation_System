## スプレッドシートのシート構成

| シート名 | 用途 |
|---|---|
| 予約 | 予約データ（読み取り: CSV公開 / 書き込み: GAS） |
| 会議室 | 会議室設定（手動編集） |
| 設定 | システム設定・管理者パスワード（手動編集） |
| ログ | 操作ログ（自動） |

## CONFIGの書き換え箇所

`index.html` と `admin.html` 両方の `const CONFIG = { ... }` を書き換えます。

```js
const CONFIG = {
  SPREADSHEET_ID: 'スプレッドシートID',   // URLの /d/【ここ】/edit
  GAS_API_URL:    'GAS Web APIのURL',       // デプロイ後のexec URL
  SHEET_GID: {
    reservations: '0',   // シートタブURLの gid=【ここ】
    rooms: '1',
    settings: '2',
  },
};
```
